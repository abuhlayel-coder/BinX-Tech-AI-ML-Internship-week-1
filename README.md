# BinX Tech AI & ML Internship — Week 1

## Overview
This repository documents Week 1 of the BinX Tech AI & Machine Learning Internship — building a reproducible Python environment and the core NumPy, Pandas, and Matplotlib skills that every later project in the program builds on. Each day's work lives in its own folder with a dedicated notebook and README, and the week closes with an integrated mini-project that ties all three libraries together into a single data pipeline.

## Highlights
- 🐍 Reproducible Python environment with a pinned `requirements.txt`
- 🔢 NumPy fluency: vectorized array creation, slicing, boolean masking, and broadcasting
- 🐼 Pandas workflow for loading, cleaning, filtering, and aggregating real-world data (Titanic dataset)
- 📊 Clear, labeled Matplotlib visualizations across line, scatter, bar, and histogram plot types
- 🧩 A Week 1 integrated mini-project combining NumPy + Pandas + Matplotlib into one load → process → visualize pipeline
- ✅ Daily, well-documented Git commits with per-day READMEs

## Week 1 Objectives
- Set up a reproducible Python environment for AI/ML development
- Build fluency with core Python fundamentals (data types, control flow, functions, OOP)
- Learn NumPy for fast, vectorized numerical computing
- Learn Pandas for loading, cleaning, and analyzing real tabular data
- Learn Matplotlib to turn data into clear, labeled visualizations
- Practice a consistent Git & GitHub workflow with daily commits
