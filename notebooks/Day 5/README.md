# Day 5 — Matplotlib

Learned how to turn cleaned data into clear, labeled visualizations — the foundation for the Exploratory Data Analysis (EDA) work ahead in Week 2.

**Objectives**
- Understand why visualization matters for both exploration and communication
- Create labeled line, scatter, bar, and histogram plots
- Combine multiple plots into one figure using subplots

**Topics Covered**
- Why visualization matters
- Matplotlib basics: `plot`, `xlabel`/`ylabel`, `title`, `show`
- The four core plot types:
  - Line (`plt.plot()`) — trend over a continuous axis
  - Scatter (`plt.scatter()`) — relationship between two numeric variables
  - Bar (`plt.bar()`) — comparison across categories
  - Histogram (`plt.hist()`) — distribution of a single variable
- Subplots (`plt.subplots()`) for side-by-side comparisons
