# Day 3 — NumPy: Numerical Computing

Learned why NumPy underpins the entire AI/ML stack and practiced the array operations everything else builds on.

**Objectives**
- Understand why vectorized NumPy arrays outperform plain Python lists
- Create, inspect, index, and slice arrays (1D and 2D)
- Use vectorized operations and boolean masking instead of loops
- Understand and apply broadcasting

**Topics Covered**
- Array creation (`np.array`, `np.zeros`, `np.ones`, `np.arange`, `np.linspace`, `np.random.rand`)
- Array attributes: `.shape`, `.dtype`, `.ndim`
- Indexing and slicing (including 2D)
- Vectorized operations and boolean masking
- Broadcasting

**Completed**

- Sliced out a specific column and row
- Used boolean masking to filter values above the array's mean
- Applied broadcasting to add a 1D array to every row of a 2D array, and verified the result manually
